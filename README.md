# wwp-data-sdk-sample 0.2.0
Open source application which demonstrates usage of WWP DATA SDK, which is a SDK for manipulating thermal images and sequences captured by Workswell Wiris Pro (WWP) and Workswell Wiris Pro SC camera.

wwp-data-sdk-sample is a desktop application developed using Qt Framework. Open .pro file in QtCreator to build or modify provided code.
